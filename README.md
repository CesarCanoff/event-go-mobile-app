### Prerequisites
- NodeJS
- npm
- Android emulator

<details>
  <summary>Simple Installation</summary>

  #### Clone the repository
```bash (SSH)
git clone git@github.com:CesarCanoff/event-go-mobile-app.git
```

#### Go to directory
```bash (SSH)
cd event-go-mobile-app
```

#### Install all dependencies
```bash (SSH)
npm install
```

#### Run the app
```bash (SSH)
npx expo start
```
  
</details>
